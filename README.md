<h1>Active Directory Homelab</h1>

<p>
This repository documents a complete Active Directory homelab built to understand
Windows authentication, DNS, and domain-based identity from a defensive
(SOC / Blue Team) perspective.
</p>

<h2>Lab Environment</h2>
<ul>
  <li>Hypervisor: VirtualBox</li>
  <li>Network: NAT Network</li>
  <li>Domain Controller: Windows Server 2022</li>
  <li>Client: Windows 10 Pro</li>
</ul>

<h2>Why This Lab</h2>
<p>
Active Directory is the backbone of enterprise Windows environments.
This lab serves as a foundation for logging, detection engineering,
and attack simulations.
</p>
